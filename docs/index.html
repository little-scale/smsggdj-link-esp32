Ableton Link → Sega Master System bridge

Flash the firmware onto a Seeed XIAO ESP32-C3 (Ableton Link over WiFi) or XIAO ESP32-S3 (Link plus USB-MIDI clock) straight from this page — no toolchain, no command line. The flasher auto-detects which board you plug in.

  1. Use Chrome or Edge on a desktop (Web Serial isn't in Safari/Firefox).
  2. Plug the board in with a USB-C data cable.
  3. Click Flash, pick the usbmodem/COM port, and wait.
Your browser can't flash — use desktop Chrome or Edge. Serial access was denied.

After flashing: WiFi setup

On first boot the bridge creates a WiFi network called SMSGGDJ-setup-XXXX. Join it from a phone or laptop; a setup page opens automatically (or browse to http://192.168.4.1). Pick your network, enter the password, and the bridge reboots onto it.

To change networks later, open the board's USB serial console and type w.

USB-MIDI clock (ESP32-S3 only)

An S3 board also shows up as a standard USB-MIDI device. Send it MIDI clock from your DAW (Start/Stop/Continue + clock) and the bridge follows it — no WiFi needed. It auto-switches to MIDI whenever clock is flowing and falls back to Link otherwise; type c link / c midi / c auto on the serial console to force it.

Open source (GPL v2-or-later, includes Ableton Link): github.com/little-scale/smsggdj-link-esp32
